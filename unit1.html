Unit 1
Primitives and
Basic Methods
Students build a programming foundation by moving from hand-drawn algorithms and grid-based coordinate systems to coding interactive sketches in Processing. Along the way, they work with variables, data types, and user input to animate robots that respond to the keyboard and mouse. A core practice throughout the unit is reading Processing.org's reference documentation to independently discover new shape functions and techniques for making sketches look exactly as intended. The unit closes with a creative robot showcase, peer code review, and an introduction to the Caesar cipher — connecting creative coding to foundational CS concepts like ASCII values and modular arithmetic.
